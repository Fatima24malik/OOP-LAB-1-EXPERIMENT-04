# Hospital Management System

A C++ Object-Oriented Programming project that manages doctors and patients using inheritance.

## Features

- Store doctor information
- Store patient information
- Display records
- Demonstrate inheritance

## OOP Concepts Used

- Classes and Objects
- Inheritance
- Base Class
- Derived Class

- Object-Oriented Programming
